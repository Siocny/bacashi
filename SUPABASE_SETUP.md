# Supabase 云同步设置指南

## 问题原因
当前同步失败是因为 Supabase 数据库中还没有创建数据表。

## 解决步骤

### 步骤 1：登录 Supabase
1. 访问 https://supabase.com
2. 登录你的账户
3. 进入项目：`twwxiucjojfujxqfaddy`

### 步骤 2：创建数据表
1. 点击左侧菜单 **SQL Editor**
2. 点击 **New Query**
3. 复制并粘贴 `supabase-setup.sql` 文件中的内容
4. 点击 **Run** 执行

SQL 内容：
```sql
-- 创建 brand_data 表
CREATE TABLE IF NOT EXISTS brand_data (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    key TEXT UNIQUE NOT NULL,
    data JSONB,
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 创建 bacashi_data 表
CREATE TABLE IF NOT EXISTS bacashi_data (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    key TEXT UNIQUE NOT NULL,
    data JSONB,
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 创建索引
CREATE INDEX IF NOT EXISTS idx_brand_data_key ON brand_data(key);
CREATE INDEX IF NOT EXISTS idx_bacashi_data_key ON bacashi_data(key);
```

### 步骤 3：验证
1. 点击左侧菜单 **Table Editor**
2. 确认能看到 `brand_data` 和 `bacashi_data` 两张表
3. 回到管理后台，点击"从云端同步"
