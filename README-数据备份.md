# 数据备份与恢复说明

## 为什么数据会丢失？

本网站使用 **localStorage（本地存储）** 保存数据，包括：
- 产品信息
- 品牌介绍
- 联系方式
- 发展历程
- 产品类别

**localStorage 的特点：**
- 数据存储在浏览器本地，不是服务器
- 清除浏览器缓存/数据时，所有数据会被删除
- 不同设备之间数据不互通
- 不同浏览器之间数据不互通

## 如何备份数据（重要！）

### 定期导出数据

1. 登录管理后台：`admin/index.html`
2. 进入 **系统设置** 或 **控制台**
3. 点击 **导出数据** 按钮
4. 浏览器会下载一个 JSON 文件（如：`cafele-data-2026-03-18.json`）
5. **将此文件保存到您的电脑或云盘**

**建议：** 每次添加或修改产品后，都导出一次备份！

## 如何恢复数据

### 方法一：导入备份文件

1. 登录管理后台
2. 进入 **系统设置** → **数据管理**
3. 点击 **导入** 按钮
4. 选择之前导出的 JSON 文件
5. 确认导入（会覆盖当前所有数据）

### 方法二：重新添加

如果之前没有备份，只能手动重新添加产品数据。

## 避免数据丢失的建议

1. **不要随意清除浏览器缓存**
   - 如果需要更新 CSS/JS，刷新版本编号即可
   - 网站已自动处理，无需手动清除缓存

2. **定期导出数据备份**
   - 每次修改后都导出一次
   - 将备份文件保存到云盘或邮箱

3. **使用固定设备管理**
   - 尽量在同一台电脑、同一个浏览器上管理数据
   - 避免在不同设备间切换

4. **手机访问提示**
   - 手机端主要用于查看，不要在手机上编辑数据
   - 手机清除缓存会导致数据丢失

## 永久解决方案

如果需要更可靠的数据管理，建议：

1. **搭建后端服务器** - 使用数据库存储数据
2. **使用云服务** - 如 Firebase、LeanCloud 等
3. **部署 CMS 系统** - 如 Strapi、Directus 等

## 当前版本功能

- ✅ 数据导出（JSON 格式）
- ✅ 数据导入（从备份文件恢复）
- ✅ 数据重置（恢复默认值）
- ⚠️ 数据存储于 localStorage（清除缓存会丢失）
